# Simple web app

To test the app locally:

1. Install Python, if not already installed
1. Install requirements:

    ```bash
    pip install -r requirements.txt
    ```

1. Set environment variables:

    ```bash
    export TEAM_NAME="<your team name>"
    export LOCATION="eu"  # Or "us"
    ```

1. Run the app:

    ```bash
    flash run
    ```
